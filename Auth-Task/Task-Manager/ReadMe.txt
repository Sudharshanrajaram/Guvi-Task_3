# User Authentication and Authorization with Bearer Token

This project demonstrates how to implement, 
user authentication and authorization using Bearer tokens in a Node.js,
application with Express.js, Mongoose, and JWT.

## Technologies Used
- Node.js
- Express.js
- MongoDB (via Mongoose)
- JSON Web Tokens (JWT)
- Bcryptjs for password hashing
